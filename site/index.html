UGK MD阅读器

给日常文档、会议记录和产品说明准备的本地 Markdown 阅读器。

下载最新版 查看功能
本地优先文件留在你的电脑上

阅读友好不是给程序员专用的编辑器

Features

从打开文件到看懂内容，少一步都不多

01

丰富 Markdown 渲染

支持表格、任务列表、删除线、Emoji、Frontmatter、代码高亮、KaTeX 数学公式和 Mermaid 图表。

02

源码对照和实时预览

需要检查原文时切到对照模式，左侧可编辑，右侧即时更新渲染结果。

03

深浅色阅读体验

明亮环境用浅色，夜间或暗色桌面用深色，窗口背景会同步切换。

04

多语言界面

内置中文、英文、日文、韩文、法文、德文、西班牙文、葡萄牙文和俄文界面。

How it works

适合电脑上的真实阅读动作

  1. 打开文件拖入 Markdown，或双击本地 `.md` 文件。
  2. 安静阅读标题、引用、代码、表格和图片保持清楚的层级。
  3. 需要时对照切到 Compare，边改源码边看右侧预览。

不像 IDE 那么重，也不像预览器那么弱。

UGK MD阅读器把 Markdown 当作普通用户每天会打开的文档来处理：速度快、界面克制、功能完整。
